# HTF Wick Sweep Order Block Indicator

## Project Overview

---

# Project Vision

The HTF Wick Sweep Order Block Indicator is an institutional liquidity framework designed to identify high-quality demand and supply zones generated by liquidity sweeps of important higher timeframe wick levels.

The indicator is based on the assumption that:

* significant HTF wicks represent institutional liquidity,
* liquidity requires time to mature,
* price sweeps that liquidity,
* strong rejection confirms institutional participation,
* the resulting imbalance creates a high-probability order block.

---

# Primary Objectives

The project has five primary objectives:

### 1. HTF Correctness

All detection occurs on the selected higher timeframe.

---

### 2. Timeframe Stability

Zones must remain identical when switching:

```text
1m
5m
15m
1H
```

charts.

---

### 3. Non-Repainting

Zones must never:

* move,
* disappear,
* relocate,
* recalculate historically.

---

### 4. Institutional Quality

Not all liquidity is equal.

The system must distinguish:

```text
Weak
Moderate
Good
Strong
Institutional
```

liquidity events.

---

### 5. Pine Performance Safety

The project must remain inside practical Pine Script limits.
